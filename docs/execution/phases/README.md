# Implementation Phases

This directory contains the bounded implementation tasks for the SignalLoop MVP.

## Rules

- Complete phases in order.
- Only implement the active phase from `CURRENT_STATE.md`.
- Mark task and acceptance checkboxes as work is completed.
- Update `CURRENT_STATE.md` before handing off.
- Record architecture changes in `docs/decisions/` only when a decision changes.
