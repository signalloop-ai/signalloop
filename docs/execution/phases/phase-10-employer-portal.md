# Phase 10: Employer Portal

## Goal

Implement minimal employer review portal.

## Tasks

- [x] Add Clerk employer login.
- [x] Create assessment list.
- [x] Create invite candidate flow.
- [x] Create candidate attempt list.
- [x] Create report detail view.

## Acceptance criteria

- [x] Employer can log in.
- [x] Employer can create invite.
- [x] Employer can view candidate report.

## Status

Complete. The web app has a Clerk-backed employer portal when Clerk keys are configured and a local development login fallback when they are not. Employers can create candidate invites, view attempt summaries, and open or generate evidence reports. The API exposes an employer attempt list with report summary metadata.
