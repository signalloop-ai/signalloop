# SignalLoop MVP PRD

## Product overview

SignalLoop is an AI-native candidate evaluator for software engineering hiring.

The MVP focuses on one backend engineering assessment using a realistic FastAPI debugging, hardening, and product tradeoff task.

## Core hypothesis

In AI-native engineering work, the differentiating skill is not whether a candidate can write code without assistance. It is whether the candidate can frame the problem, use AI effectively, verify AI-assisted output, make reasonable engineering decisions, and own the final solution.

## Evaluation framework

FAVO = Frame, Ask, Verify, Own.

- Frame: understand problem, constraints, risks, and ambiguity.
- Ask: use AI through focused questions rather than over-delegation.
- Verify: run tests, add tests, inspect edge cases, and validate assumptions.
- Own: explain final changes, decisions, tradeoffs, and remaining risks.

## MVP scope

In scope:

- one preloaded backend assessment,
- candidate invite link,
- web coding workspace,
- Monaco editor,
- public test execution,
- constrained AI assistant,
- final submission,
- hidden evaluation,
- Engineering Evidence Report,
- employer review portal.

Out of scope:

- video proctoring,
- ATS integration,
- enterprise SSO,
- multi-language assessment marketplace,
- production billing.
