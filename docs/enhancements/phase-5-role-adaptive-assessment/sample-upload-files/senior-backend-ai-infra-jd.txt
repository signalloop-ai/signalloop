Company: SignalLoop SampleCo

Role Overview
We are hiring a Senior Backend Engineer for an AI infrastructure team. The role requires Python, FastAPI, API design, authorization, multi-tenant APIs, reliability, observability, PostgreSQL, Kubernetes basics, and strong AI collaboration.

The team builds internal control-plane APIs for provisioning model-serving workloads. We need engineers who can debug production behavior, protect tenant boundaries, write useful tests, and make safe product tradeoffs under ambiguity.

Responsibilities
- Build and maintain Python FastAPI services for internal platform workflows.
- Design predictable REST APIs with clear validation, error handling, and authorization behavior.
- Protect tenant boundaries across teams and organizations.
- Write meaningful pytest coverage for public behavior, edge cases, and regressions.
- Collaborate with platform engineers on deployments, monitoring, and Kubernetes-operated services.
- Use AI tools responsibly for debugging, test design, and implementation support while verifying outputs.

Required Skills
- Python 3 and FastAPI

- Input validation, state transitions, and error handling
- PostgreSQL-backed workflows
- Reliability, observability, and production debugging
- Pytest and regression test design
