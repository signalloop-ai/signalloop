# Advanced v1 Scoring Rubric

Use the Phase 2 100-point SignalLoop rubric:

| Category | Points | Advanced evidence focus |
|---|---:|---|
| Public issue resolution | 15 | Initially failing public tests fixed without breaking starter-pass behavior. |
| Private issue generalization | 20 | Hidden edge cases across membership, authorization, partial update, archive, audit, comments, and lists. |
| Feature/design implementation | 20 | Team lead permissions, archive semantics, unauthorized behavior, audit behavior. |
| Candidate-written tests | 15 | Meaningful tests for authorization, partial updates, archive/list behavior, and events. |
| AI collaboration | 20 | Focused AI use without full-solution or defect-enumeration requests. |
| Regression/code quality | 10 | Existing API behavior remains stable and maintainable. |

Do not give a separate written-explanation score. Submission Review answers are
supporting evidence for ownership, verification, and follow-up questions.
